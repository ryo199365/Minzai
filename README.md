# Minzai 在庫管理アプリ　機能概要（途中）

# 🏠 トップ画面

- URL: `/`
- 目的: 各機能への入口、在庫警告を表示

### 主な要素

| 要素名 | 種類 | 説明 |
|--------|------|------|
| 在庫警告リスト | 表 / カード | 在庫数が一定以下のアイテムを表示 |
| 入庫ボタン | ボタン | 入庫一覧画面へ遷移 |
| 出庫ボタン | ボタン | 出庫一覧画面へ遷移 |
| 新規登録ボタン | ボタン | 新規登録画面へ遷移 |
| 削除ボタン | ボタン | 削除画面へ遷移 |
| 履歴ボタン | ボタン | 履歴画面へ遷移 |
| 在庫確認ボタン | ボタン | 在庫確認画面へ遷移 |

# 📦 入庫画面

- URL: `/in`
- 目的: 入庫するアイテムを選択、数量を入力し在庫更新

### 入庫一覧画面

| 要素名 | 種類 | 説明 |
|--------|------|------|
| アイテム検索 | テキスト入力 | 名前やコードで絞り込み |
| ジャンルボタン | ボタン | 例：「衣類」「食品」「備品」で絞り込み |
| アイテム一覧 | 表 / カード | 名前 + 在庫数表示。クリックで個別入庫画面へ |
| 戻るボタン | ボタン | トップ画面へ戻る |

### 個別入庫画面

| 要素名 | 種類 | 説明 |
|--------|------|------|
| 現在在庫数 | 表示 | 選択アイテムの在庫 |

# ➕ 新規登録画面

- URL: `/register`
- 目的: 新しいアイテムを追加

| 要素名 | 種類 | 説明 |
|--------|------|------|
| アイテム名入力 | テキスト | 新規アイテム名 |
| ジャンル選択 | プルダウン | 備品 / 薬 / 飲み物 |
| 新ジャンル追加 | テキスト | 新しいジャンルを追加する場合に入力 |
| 初期在庫数 | 数値入力 | 初期在庫 |
| 画像アップロード | ファイル | アイテムの画像を選択(任意) |
| 決定ボタン | ボタン | 登録処理 → 履歴追加 |
| 戻るボタン | ボタン | トップへ戻る |

# 🔄既存アイテム編集画面

- URL: `/edit` （検索・選択画面）
- URL: `/edit/<id>` （個別編集画面）
- 目的: アイテム名・ジャンル・在庫数・画像を更新

| 要素名 | 種類 | 説明 |
|--------|------|------|
| 検索ボックス | テキスト入力 | 名前やコードで部分一致検索 |
| ジャンルボタン | ボタン | カテゴリでフィルタ |
| アイテム一覧 | 表 / カード | 名前 + 在庫数 + 警告表示 クリックで個別編集画面へ|
| 編集フォーム | フォーム | アイテム選択後、名前・ジャンル・在庫・画像を変更可能 |
| 保存ボタン | ボタン | 押すと確認メッセージ → 更新して一覧へ戻る |
| 戻るボタン | ボタン | トップ画面へ戻る |
